Steps to Execute 
=================
1. Import the Project in eclipse.
2. Right Click on Run.xml-> Select TestNG Suite

Executes Test cases one by one.
While executing TestCase2.Java
Please select the image manually when captcha Checkbox is selected. (Given Thread.sleep(7000) wait time to select image)

To View Report
==============
After Completion of Execution, follow the steps

1. Open the test-output folder
2. Go to test-output\html\index.html
3. Open in Browser

To View Result Code of each method in Excel Sheet
==============
1. Drive Name:Location of Project\Sogeti_Stuttgart\Project_App\sogetiapp1\src\test\resources\TestData
2. Open resultstatus.xlsx
